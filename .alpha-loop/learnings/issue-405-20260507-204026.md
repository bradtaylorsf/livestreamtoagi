## Patterns
- Pair a persistence-layer test with a route-layer test to cover both the storage contract and the API field-mapping contract independently.
- When adding new DB-backed UI tabs, include a backfill script alongside the migration so historical data populates the UI without re-running simulations.
- Validate user-supplied file paths via `Path.resolve().relative_to(allowed_dir)` to reject traversal and absolute paths in one check (used for `seed_file` validation).

## Anti-Patterns
- Inconsistent error containment across code paths: phased-mode wraps assertion evaluation in try/except but autonomous-mode (orchestrator.py:841-851) does not. New checkpoint hooks should match existing error-handling conventions in sibling code paths.
- Backfill scripts that require manual invocation risk leaving acceptance criteria ("UI immediately shows non-empty results") technically unverified at PR time — note the manual step explicitly in the PR.
