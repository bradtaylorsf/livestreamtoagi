## Patterns
- Place FastAPI app in `core/main.py` rather than root `main.py` to match the project's package structure
- Use `[project.optional-dependencies]` for dev deps rather than mixing them into main dependencies
- Include directory existence tests (`test_structure.py`) as a cheap smoke test for project scaffolding tasks

## Anti-Patterns
- Maintaining both `requirements.txt` and `pyproject.toml` separately — risks dependency drift; generate one from the other
- Leaving `uv.lock` untracked when using `uv` as package manager — loses reproducible installs
