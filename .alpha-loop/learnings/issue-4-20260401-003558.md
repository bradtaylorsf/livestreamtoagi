## Patterns
- Mock FastAPI lifespan dependencies (db, redis) in unit tests to avoid requiring Docker services
- Use `asynccontextmanager` lifespan pattern over deprecated `on_event` decorators for FastAPI startup/shutdown
- Graceful codec registration (catch `ValueError` for missing pgvector extension) prevents startup failures in dev environments
- Health endpoints should report per-dependency status and return "degraded" on partial failure

## Anti-Patterns
- Module-level `TestClient(app)` instantiation triggers lifespan hooks at import time — always create test clients inside test functions or fixtures
- Defining Pydantic models that no code path uses (dead models like `CoreMemoryUpsert`) — only create models as needed by repo method signatures
- Using `Any` type for JSONB columns (`participating_agents`, `topics_discussed`) instead of specific types like `list[str]`
